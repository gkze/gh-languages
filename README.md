# `gh trending`: a [`gh` CLI](https://cli.github.com/) extension to show trending repos

`¯\_(ツ)_/¯`

## Installation

If you already have [`gh`](https://cli.github.com/) installed, then run:
```bash
gh extension install gkze/gh-languages
```
